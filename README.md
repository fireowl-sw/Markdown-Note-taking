# Markdown 笔记应用

一个简洁、高效的在线 Markdown 笔记工具，支持实时预览、笔记管理和本地存储。

## 功能特点

- ✨ **实时预览** - 编辑时即时显示 Markdown 渲染效果
- 📝 **笔记管理** - 创建、编辑、删除笔记
- 💾 **本地存储** - 数据存储在浏览器 LocalStorage
- 🔍 **搜索功能** - 快速搜索笔记标题和内容
- 🎨 **代码高亮** - 支持多种编程语言的语法高亮
- ⌨️ **快捷键** - 支持 Tab 缩进和 Ctrl+S 保存

## 技术栈

- React 19
- TypeScript 5
- Vite 6
- Tailwind CSS 3
- React Markdown
- React Syntax Highlighter

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 使用

1. 点击「新建笔记」创建新笔记
2. 在中间编辑器输入 Markdown 内容
3. 右侧实时查看渲染效果
4. 使用搜索框快速查找笔记
